# tic-tac-toe
#️⃣ Projeto da disciplica de Técnicas de Programação


## Features

- Ranking
- Credits
- Multiplayer


## Requirements
  * Windows
  * [gcc](https://gcc.gnu.org/)

## Instalation



```bash
  git clone https://github.com/pdr-tuche/tic-tac-toe.git
  cd tic-tac-toe/src
  gcc -o exec.exe main.c
```
